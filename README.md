# Vue Vite Pinia Boilerplate

Opinionated Vue Boilerplate.

### Stacks
- Vue 3
- Vite
- Vue-router
- Pinia
- Jest
- Vue-test-utils
- Typescript

### UI
- Ant-design-vue
- Tailwindcss

### Tools
- EsLint AirBnb
- Husky

### Installing
Use this template

Run `npm install`

Run `npm run dev`

### Customizing Ant design style
Use `/src/assets/styles/variable.less` to customize Ant design theme.
